Name
TwoLayerNet
====

Overview
 二層ニューラルネットのフルスクラッチ実装です。MNISTのデータを学習・分類し、accuracyの推移をグラフで表示します。

## Requirement
numpy
matplotlib

## Usage
実行するだけ。コンソールに

MNIST dataset loaded.
TwoLayerNet initialized.
progress: 0/10000
train_acc, test_acc | 0.09841666666666667, 0.098

と表示されれば動いています。
